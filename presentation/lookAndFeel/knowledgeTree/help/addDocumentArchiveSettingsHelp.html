Add Document Archive Settings

This page allows you to create archiving settings for a document:

  1. Archiving Type: Choose whether this document should be archived based on date/time settings or on user utilisation.
  2. Archive By Date:
    1. Either enter an expiration date ie. when the document will be archived.
    2. Or specify a period from the document creation date ie. 3 Months after creation.
  3. Archive By Utilisation:
    1. Specify the document transaction that will trigger document archival.
    2. Enter the period after which the document will be archived. (Eg. If you select the Download transaction and 3 days as the period, then the document will be archived 3 days after the last Download.).
  4. The Clear button resets the period and date information.
  5. The Submit button creates the specified archiving settings.
  6. The Back button returns you to the archiving type selection page.
  7. The Cancel button returns you to the document details page.
