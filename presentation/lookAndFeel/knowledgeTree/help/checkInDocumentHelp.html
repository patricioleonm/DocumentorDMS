‘Check-in Document

  1. Enter the path of the document you wish to check in or use the Browse button to locate the document.
  2. Enter a comment for the checked-in document (mandatory).
  3. Select whether the changes made to the document were minor or major by clicking on Minor CheckIn or Major CheckIn respectively. This selection affects the document version number (see the Document Data table).
  4. Click the Check-In button to upload the file or Cancel to cancel the check-in.
