Add Document Field Look-up Data

  1. Select the document field from the drop-down list. If the field does not appear as expected then it has not been designated as a look-up field. To make the field a look-up field, go to the 'Edit a Document Field' section, select the field, then check the 'Has Lookup' box.
  2. Enter a new lookup value in the text box and click Add to save this value, or Cancel to discard it.
