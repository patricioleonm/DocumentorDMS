System Settings

The system settings are as follows:

LDAP Server:

Directory server for user authentication

LDAP Root DN:

The root from which users are found in the LDAP server

Email Server:

SMTP server used for all outgoing mail

Administrator Email Address:

<your e-mail address>

Administrator Name:

<your name>

Default E-mail Address:

Default address from which notifications are sent

Default E-mail Name:

Default “From” name used in the notification system

File System Root:

The directory in which the DMS is installed

Document Root:

The directory in which all documents are stored

Language Directory:

The directory that contains all the language resource templates

User Interface Directory:

The location of the presentation and business logic files

Root URL:

The DMS base URL

Graphics URL:

The location of graphic files used in the DMS

User Interface URL:

The location of the presentation and business logic pages

Use File System:

Whether to use file system or database to store documents

Default Language:

The language used if no alternative is specified

Session Timeout:

Session timeout length in seconds
